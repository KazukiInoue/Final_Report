# I am Sugar  

You are a cube sugar.  
Escape from ants!  
Go to the salt mountain!!

## How to play    
* Left/Right key   : move to the left/right
* Up/Down key      : go/come
* WHITE/BLACK mode : click the WHITE/BLACK image
* ???? mode        : click somewhere...
